> **Skill-first, MCP-second** bridge from Claude Code / OpenAI Codex to **Google Antigravity CLI** (`agy`).
> Inherits the operational discipline of [`upstream/reference`](https://example.invalid/upstream) and
> [`upstream/reference`](https://example.invalid/upstream), and extends them
> with capability detection, async long-task supervision, dual-backend routing (agy + gemini-cli compatible fallback),
> safe-by-default execution, Codex/Antigravity skills, and a stable stream-json-compatible event schema.

Full design, architecture, capability detection, output strategy and safety model live under [`docs/`](docs/).

This README is a stub during scaffolding — Phase 8 expands it into the full installation / usage / safety guide.
